# Zuna Pet Essentials

E-commerce website for pet essentials.

## Installation

1. Install dependencies:
   ```bash
   pnpm install
   ```
   Or use `npm install` / `yarn install`.

2. Start the dev server:
   ```bash
   pnpm dev
   ```

3. Build for production:
   ```bash
   pnpm build
   ```
